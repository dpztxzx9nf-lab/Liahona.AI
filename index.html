Liahona

A grounded conversational system for KINDEX.

Liahona is an early Discord-based system for interpretation, response judgment, and future source-grounded retrieval across a human archive.

Current System

Discord bot

Receives messages, interprets intent, generates replies, and delivers responses.

PM2 runtime

Runs as a managed Node process with simple restart and log commands.

Observability

Structured lifecycle logs show where each message succeeds or fails.

DM support

Direct messages are part of the working surface and need clear diagnostics.

Semantic retrieval direction

Retrieval will become more precise as memory and source indexing mature.

Source grounding direction

Future answers should point back to trusted source material when possible.

Lifecycle

  1. MESSAGE_RECEIVED
  2. INTERPRETATION_RESULT
  3. GENERATION_RESULT
  4. DELIVERY_RESULT

Operations

pm2 restart liahona --update-env
pm2 logs liahona
pm2 logs liahona --err
pm2 logs liahona --out

Architecture Direction

KINDEX is the human archive.

Database is memory and retrieval.

Sources are grounding.

Liahona is the judgment and interpretation layer.
