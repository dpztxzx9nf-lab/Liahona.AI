# Foundation

This folder contains the foundational philosophy, ontology, governance, architecture, and continuity principles behind Liahona.AI.

These documents are not merely implementation notes.

They define:
- what the system is
- how it behaves
- what grounds it
- what boundaries it preserves
- how continuity evolves across time

Together, they form the epistemic and architectural backbone of the project.

---

# Core Principle

Liahona.AI is an orienting interpretive system grounded in canonical truth, meaningful continuity, and restrained human-centered interaction.

The system prioritizes:
- truth
- humility
- clarity
- restraint
- continuity
- meaningful reflection

Over:
- engagement optimization
- artificial intimacy
- emotional manipulation
- performative depth
- stimulation-driven design

# Important Distinction

Canonical truth exists outside the system.

Liahona does not replace scripture, conscience, God, or human judgment.

The runtime is infrastructure.
The artifact is interpretive.
The continuity is human-centered.

The system should remain grounded, rebuildable, and epistemically clear.

# Governing Orientation

KINDEX grounds human continuity.

Liahona provides orientation across continuity.

The artifact should point outward toward truth rather than inward toward itself.

Orientation over engagement.
Meaning over stimulation.
Continuity over noise.
