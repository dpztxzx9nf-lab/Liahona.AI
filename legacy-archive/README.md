# Legacy Archive

This folder preserves historical continuity and reference material for Liahona.AI.

It is not active governance and should not be treated as the current source of runtime truth.

## Purpose

The archive keeps prior foundation drafts, exploratory notes, historical source stubs, continuity summaries, and product-direction artifacts available for reference.

Use it to understand how the project evolved, not to decide what the current runtime does.

## Active Replacements

- Use `core/` for active governance and ontology.
- Use `.cursor/rules/` for active agent guidance.
- Use `STATUS.md` for implementation status.
- Use `docs/GUIDE.md` for current runtime operation.
- Use `continuity/` for curated semantic checkpoints.

## Handling

Do not delete, rename, or move archive material casually.

If archive content is useful, cite or summarize it in active documentation rather than merging it back wholesale.
