# Docs

This folder contains supporting implementation and source documentation for Liahona.AI.

Use this folder for operational guides, data planning, source references, and implementation notes that support the active runtime and future development.

## Current Files

- `GUIDE.md` is the main runtime and PM2 operation guide.
- `DATA.md` is planning-only documentation for future data, memory, and reset scopes.
- `Liahona.AI.md` describes the broader product and experience direction.
- `Eternal Sources/` contains outward source references used for grounding and source planning.

## Status Boundary

Docs may describe planned systems, but planned systems are not current runtime behavior.

Memory, embeddings, SQLite, retrieval, source ingestion, and frontend artifact UI should be treated as planned unless `STATUS.md` says otherwise.
