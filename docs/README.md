# Liahona.AI Documentation

This folder is the lightweight project memory for Liahona.AI.

Use it to keep architecture, continuity, operations, prompts, roadmap, and source-grounding notes readable without turning the static site into the documentation system.

## Folders

- `architecture/` - design philosophy and system shape.
- `continuity/` - versioning, checkpoints, and rollback notes.
- `operations/` - commands, deployment workflow, and safe update steps.
- `prompts/` - future prompt notes and prompt behavior references.
- `roadmap/` - near-term and later direction.
- `sources/` - grounding source lists and source policy notes.
