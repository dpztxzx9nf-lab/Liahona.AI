# Operational Commands

Run these from the Liahona.AI repository:

```powershell
cd C:\Projects\Liahona.AI
git status
```

Static syntax checks:

```powershell
node --check src\cosmology.js
node --check src\dashboard.js
node --check script.js
```

Diff checks:

```powershell
git diff --check
git diff --stat
```

Commit and push:

```powershell
git add .
git commit -m "Describe change"
git push origin main
```
