# Roadmap

Near-term:

- Keep the homepage artifact-first.
- Refine the mobile continuity device as a secondary layer.
- Improve artifact motion, depth, and grounding interactions.
- Keep source and continuity data modular.

Later:

- Retrieval refinement.
- Journal continuity.
- Weekly digest.
- Lightweight dashboard views behind the continuity device.
- Backend continuity storage.
- Authentication only when there is a clear private data need.

Avoid:

- Heavy 3D engines before the interaction model is stable.
- Dashboard-first first-load experience.
- Large backend architecture before static-site behavior is clear.
