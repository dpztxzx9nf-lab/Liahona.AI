# Liahona.AI

## Core Orientation

Liahona.AI is an artifact-first orienting continuity system centered on:

* grounding
* reflection
* continuity
* symbolic interaction
* truthful outward orientation

It is not merely:

* a chatbot
* a dashboard
* a productivity app
* an engagement platform
* a virtual world
* a gamified system

The goal is not maximum engagement.

The goal is orientation.

* Bible
* Book of Mormon
* Doctrine and Covenants
* Pearl of Great Price

* Guide to the Scriptures

* cosmological framing
